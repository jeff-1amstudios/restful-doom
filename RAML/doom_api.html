    Angle this object is pointing

    Identifier of this object

    Example:

    1

    Health remaining

  • armor: (integer)

    amount of armor remaining

  • position: (object)

    Location of this item in world units

    • x: (integer)
    • y: (integer)
    • z: (integer)

      positive z points upwards

  • distance: (integer)

    Distance from player

  • items: (integer)

    number of items picked up

  • flags: (object)
    • MF_SPECIAL: (boolean)

      When this item is picked up, call P_SpecialThing

    • MF_SOLID: (boolean)

      Cannot be walked through

    • MF_SHOOTABLE: (boolean)

      Takes damage when shot

    • MF_NOGRAVITY: (boolean)

      Don't apply regular world gravity

    • MF_NOCLIP: (boolean)
    • MF_SHADOW: (boolean)

      Render this item with the flickering half-invisible shadow rendering

    • MF_CORPSE: (boolean)

      Don't respond to any action

  • attacking: (integer)

    ID of the object this object is attacking

/api/player/actions post

post

Create an action for the player to perform (RESTRICTED)

post /api/player/actions

Create an action for the player to perform (RESTRICTED)

  • Request
  • Response

Body

Media type: application/json

Type: object

Properties
  • type: required (string)

    Action to perform

    Example:

    shoot | forward | backward | turn-left | turn-right | use | strafe-left | strafe-right | switch-weapon
  • amount: required (integer)

    Amount to move or turn (ignord for shoot and use types), or weapon to select. Defaults to 10 if omitted

    Example:

    25

Example:

{
  "type": "strafe-left",
  "amount": 25
}

HTTP status code 201

HTTP status code 400

Invalid parameters

Permission denied. Multiplayer clients may not perform this action

HTTP status code 200

Body

Media type: application/json

Type: object

Properties
  • id: (integer)

    Identifier of this object

    Example:

    1
  • position: (object)

    Location of this item in world units

    • x: (integer)
    • y: (integer)
    • z: (integer)

      positive z points upwards

  • angle: (integer)

    Angle this object is pointing

  • health: (integer)

    Health remaining

  • type: (string)

    Type of this object

  • distance: (integer)

    Distance from player

  • attacking: (integer)

    ID of the object this object is attacking

  • flags: (object)
    • MF_SPECIAL: (boolean)

      When this item is picked up, call P_SpecialThing

    • MF_SOLID: (boolean)

      Cannot be walked through

    • MF_SHOOTABLE: (boolean)

      Takes damage when shot

    • MF_NOGRAVITY: (boolean)

      Don't apply regular world gravity

    • MF_NOCLIP: (boolean)
    • MF_SHADOW: (boolean)

      Render this item with the flickering half-invisible shadow rendering

    • MF_CORPSE: (boolean)

      Don't respond to any action

Example:

{
  "id": 1
}
